data SymEqualTo c Source #

Predicate equality for symbols.

Char Predicates

This section models the boolean functions defined in Data.Char. See that module for more information regarding these definitions.

Although the instances are defined for Char, they can be extended to String and Text via Refined.Predicates.Foldable's All.

Examples

Expand
>>> validate @(All Alpha) Proxy "abc"
Nothing

>>> showRefineException <$> validate @(All Alpha) Proxy "abc1"
Just "RefineOtherException (Alpha) \"1 is not an alphabetic character\""
